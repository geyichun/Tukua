# 图快

一个给非技术同学用的图片处理工具箱。打开浏览器就能用，不需要装任何东西。

## 这东西能干嘛

把工作中跟图片相关的杂活全包了：

- 图片太大微信发不了？拖进来压一下，自动处理好
- 领导要你给产品图加水印？批量拖进去，一键全搞定
- 设计给了 PNG 你要转 JPG？不用喊开发帮忙，自己来
- 多张图要拼成一张？上下左右随便拼
- 图太大了要裁一下？拖进来按尺寸裁剪

总之就是：**跟图片有关的重复劳动，打开图快就完事。**

## 怎么用

1. 打开在线地址：（部署后填这里）
2. 或者把代码下载下来，双击 index.html 就能用。不需要 Node、不需要 Python、不需要装任何东西。

## 为什么做了这个

在外包公司呆了四年，见过太多运营妹子为了改一张图装 Photoshop 装一个下午，也见过产品同学为了压缩图片到处找在线工具结果全是广告。市面上不是没有图片处理工具，是它们要么太复杂（Photoshop），要么广告太多（各种在线站），要么要花钱。

图快的原则很简单：

- 一步到位，打开就用
- 没广告，不收费
- 所有处理都在你浏览器里跑，不上传服务器，隐私安全
